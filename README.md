## 准备

``` bash
# 安装依赖
npm install

# 开发模式 地址http://localhost:8080
npm run dev

# 生产打包
npm run build
```

## 特点

* 使用 svg sprite 代替 css sprite
* CSS Grid & Flexbox
* 使用 Stylus 代替 Sass
* 待补充

因为 CSS Grid 的缘故只兼容 firefox 52/chrome 57/safari 10.1/edge 41 及以上, [demo](https://lluvio.github.io/vue-element/#/auth)
